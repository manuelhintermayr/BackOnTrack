To start the program, go into the BackOnTrack-folder and open "BackOnTrack.sln".
You will need Microsoft Visual Studio 2017 (tested with Version 15.8.7), .NET Framework 4.6.2 and a internet connection to build and run the programm.
In the "BackOnTrack" solution please start the "BackOnTrack" project.


The folder "A2A.TAF" contains the first sample implementation for UI testing.
